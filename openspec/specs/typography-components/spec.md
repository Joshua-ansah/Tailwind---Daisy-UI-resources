## ADDED Requirements

### Requirement: Typography scale and patterns are shown
The `components/typography.html` page SHALL display typographic elements and prose patterns.

#### Scenario: Heading scale is shown
- **WHEN** the page renders
- **THEN** it SHALL show h1 through h6 with size and weight labels

#### Scenario: Body text, lead, muted, and small variants are shown
- **WHEN** the page renders
- **THEN** it SHALL show paragraph variants including lead text, muted helper text, and small/caption text

#### Scenario: Blockquote, code, and kbd elements are shown
- **WHEN** the page renders
- **THEN** it SHALL show a styled blockquote, an inline code snippet, a code block, and a `<kbd>` keyboard shortcut

#### Scenario: Lists and dividers are shown
- **WHEN** the page renders
- **THEN** it SHALL show ordered, unordered, and description lists, plus a DaisyUI divider
