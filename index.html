About this Project

Personal Collection

Hand-curated UI components and layout patterns I use as a starting point in my own projects. Everything is built and tested with TailwindCSS v4 and DaisyUI.

AI-Ready Templates

Designed to be referenced by AI coding assistants. Each component page is a self-contained, readable HTML file — easy for a model to parse, adapt, and build on top of.

No Framework Lock-in

Plain HTML files served by Vite. No React, no Vue, no build complexity — just utility classes and components you can copy directly into any project.

Stack

TailwindCSS v4

Utility-first CSS — loaded via the @tailwindcss/vite plugin. No PostCSS config needed.

DaisyUI

Semantic component classes on top of Tailwind. Loaded as a Tailwind plugin with @plugin "daisyui".

Vite v7

Instant dev server with HMR. Run npm run dev to start locally.

Plain HTML

No framework, no JSX. Each component is a standalone .html file — easy to read, copy, and hand to an AI.

This is an independent personal project. I am not affiliated with, endorsed by, or connected to DaisyUI or Tailwind Labs in any way.
